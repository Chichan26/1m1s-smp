How to Join

Follow these steps to join 1M1S Network

1

Join our Discord Server

Connecting to our Discord server is mandatory. All community updates and account management happen here. Click the link to join below.

Click here!
2

Read the Rules

Familiarize yourself with the regulations in the #rules channel. These are strictly enforced to ensure a fair experience for everyone.

3

Connect to the Server

Launch Minecraft Java Edition and use the following server IP to join: 1m1s.network

4

Link Your Account

Upon joining the Minecraft server, you will be prompted with a unique link account code.

• Copy the code from your Minecraft chat.

• Head to the #Link-Account channel in Discord.

• Enter the code there to verify your identity.

5

Play and Enjoy!

Once linked, you are all set. Welcome to the community! 🎮
